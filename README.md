## Convention
* Github Workflow 사용
* 각자 필요한 Branch 생성
* 각 Chapter 및 item 별 패키지에 itemTitle_GithubUserName 으로 저장
* 작성 완료 후 reviewer 지정 후 main 으로 PR
* 질문 및 필요사항 review 후 merge
